Class Messaging​Factory 

@objc(ZDKMessagingFactory) public final class MessagingFactory: NSObject

This factory is the starting point to begin using Messaging. It can be used to create instances of Messaging, which is the main entry point of the SDK.

create(with:​completion​Handler:​) 

public func create(with credentials: MessagingCredentials, completionHandler: @escaping (Result<Messaging, MessagingError>) -> Void)

This method is used to create or retrieve a Messaging instance. It is an asynchronous call that retrieves settings from the network based on the MessagingCredentials it has been passed.

If an integrator calls this method again with the same MessagingCredentials, a network request will not be made and it will instead return the same instance that was initialized upon the first call. If an integrator calls this method again with new MessagingCredentials, a network will be made and if successful, it will return a new Messaging instance. Integrators will not be able to retrieve the first Messaging instance from this method once a new one has been created and returned.
